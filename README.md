# Drag & Drop Custom Sort

A VS Code extension that allows you to **manually reorder files** in the explorer using drag and drop, regardless of alphabetical order.

## Features
* **Custom Sorting:** Organize your files in the order *you* want.
* **Intuitive Drag & Drop:**
    * **Insert:** Drag file A *over* file B to insert A **before** B.
    * **Move to End:** Drag a file onto a **Folder** (or empty space) to move it to the **bottom** of the list.
* **Persistence:** Your custom order is automatically saved in `.vscode/sort-order.json`.

## How to Use
1.  Open the **Drag & Drop** view in the Explorer sidebar.
2.  Simply drag and drop files to rearrange them!

## Requirements
* VS Code ^1.108.0 or newer
